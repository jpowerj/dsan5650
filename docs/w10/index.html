Instrumental Variables

If randomization works to obtain causal effects‚Ä¶

‚Ä¶Find something random in the causal system, use e.g. matching to ‚Äúforce‚Äù the same scenario!

General form: \(\text{Effect}(D \rightarrow Y) = \frac{\text{Effect}(Z \rightarrow Y)}{\text{Effect}(Z \rightarrow D)}\) (Try ‚Äúplugging in‚Äù \(Z\) = Coin Flip!)

\[ \beta_{\text{IV}}^{\text{Wald}} = \frac{ \mathbb{E}[Y_i \mid Z_i = 1] - \mathbb{E}[Y_i \mid Z_i = 0] }{ \mathbb{E}[D_i \mid Z_i = 1] - \mathbb{E}[D_i \mid Z_i = 0] }, \; \beta_{\text{IV}} = \frac{\text{Cov}[Y, Z]}{\text{Cov}[D,Z]} \]

Text-as-Data Part 1: TAD in General

  • Computers don‚Äôt exactly ‚Äúread‚Äù text! They process numeric representations of some feature(s) of the text
    • Ex: sentiment, topic, embedding in semantic space
  • \(\Rightarrow\) When we do causal inference with text, we‚Äôre not studying \(D \rightarrow Y\) itself! Instead, we study:
    • Text as Outcome: \(D \rightarrow g(Y)\) and/or
    • Text as Treatment: \(g(D) \rightarrow Y\)
