  • In e.g. 5300, you learned a bunch of ad hoc models: Linear Regression, Decision Trees, SVMs, Neural Nets
  • PGMs provide a language we can use to tell a computer how to estimate parameters in these models from data

Now → August

  • The class splits into two parallel goals!
  • What kinds of cool computational social science models are unlocked, that we can now implement in this language?
  • How does this language provide a scaffolding for causal inference

Why Take the Time to Learn a Modeling Language (vs. Individual Models)?

  • My answer: allows you to adapt to specifics/idiosyncrasies of your problem!
  • The “modeling language” metaphor: Learning models vs. learning modeling language \(\Leftrightarrow\) Learning phrases in a language vs. learning to speak the language
  • “Hello, one hamburger please” is good, but what happens when you need to remove pickles? Or you want to add a sauce?
  • McElreath (2020) example: How do I “tell” my model to (a) Capture specific effects of tanks, but (b) pool information about tadpoles across tank sizes?

Example: Multilevel Modeling

Nightmarish Without a Modeling Language!

  • Modeling language \(\Rightarrow\) Unambiguously “encode” idiosyncratic domain knowledge
  • Dissertation: Cold War \(\times\) “Third World” \(\leadsto\) Cuban 🇨🇺 trans-continental operations1
  • Main narrative (for estimation): 1975 (South Africa invades Angola, 14 Oct → 🇨🇺 intervention, 4 Nov) to 1979 (USSR requests 🇨🇺 troops to Ethiopia for Ogaden War)
  • [Ontology] Fix 1979 geographic entities at National level (as modeling choice, like fixing 2000 USD to measure inflation): \(\textsf{Cuba}_{1979}\), \(\textsf{Angola}_{1979}\), \(\textsf{PDRY}_{1979}\), \(\textsf{YAR}_{1979}\)
  • Different tokens (Think NLP: "Congo", "DRC", "Republic of Congo") can then be contextualized: can “track” and link data appropriately despite splits, merges, name changes
  • Say we have data on “Number of Communist Militants in \(X\)” (Hoover Yearbook)…

The Ladder of Causal Inference

Counterfactuals: What would have happened, if history was slightly different…
\(\Pr(Y_{M=M_0} \mid \textsf{do}(X)) - \Pr(Y_{M=M_0} \mid \textsf{do}(\neg X))\)		 Modding a videogame
↑
Intervention: What happens if I…
\(\Pr(Y \mid \textsf{do}(X)) - \Pr(Y \mid \textsf{do}(\neg X))\)		 Playing a videogame
↑
Association: What happened?
\(\Pr(Y \mid X) - \Pr(Y \mid \neg X)\)		 Watching someone play a videogame
