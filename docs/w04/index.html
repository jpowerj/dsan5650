The Ladder of Causal Inference

Counterfactuals: What would have happened, if history was slightly different…
\(\Pr(Y_{M=M_0} \mid \textsf{do}(X)) - \Pr(Y_{M=M_0} \mid \textsf{do}(\neg X))\)		 Modding a videogame
↑
Intervention: What happens if I…
\(\Pr(Y \mid \textsf{do}(X)) - \Pr(Y \mid \textsf{do}(\neg X))\)		 Playing a videogame
↑
Association: What happened?
\(\Pr(Y \mid X) - \Pr(Y \mid \neg X)\)		 Watching someone play a videogame
